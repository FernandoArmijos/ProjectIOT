Puesta en marcha en un entorno de desarrollo Django

Requisitos previos

Nota Importante: Necesita completar cada uno de los pasos para que el proyecto sea levantado correctamente en su entorno de desarrollo.

1. Tener Python 3 instalado:
python --version
2. Tener Git instalado:
git --version
3. Tener pip y venv instalados:

En la mayoría de instalaciones de Python vienen incluidos.

Pasos para levantar el proyecto

1. Clonar el repositorio del proyecto (si está en GitHub):
git clone https://github.com/FernandoArmijos/ProjectIOT.git
cd tu-repo-ProjectIOT
2. Crear y activar entorno virtual:
python -m venv venv
# En Windows
venv\Scripts\activate
# En Linux/macOS
source venv/bin/activate
3. Instalar dependencias:
pip install -r requirements.txt
4. Migrar la base de datos:
python manage.py migrate
5. Crear un superusuario (opcional):
python manage.py createsuperuser
6. Correr el servidor:
python manage.py runserver

Luego de ejecutarlo, acceda desde su navegador a: http://127.0.0.1:8000/

Verificar que funcionen:
  • El panel de administración: /admin/
  • El dashboard o interfaz web
